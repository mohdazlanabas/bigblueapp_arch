2.2 Cloud Infrastructure (Google Cloud Platform)

The entire Big Blue SuperApp infrastructure will be deployed on Google Cloud Platform, leveraging GCP's enterprise-grade services. The primary region will be asia-southeast1 (Singapore) with disaster recovery provisions in asia-southeast2 (Jakarta).

Why Google Cloud Platform?
GCP offers superior managed Kubernetes (GKE), best-in-class networking infrastructure, native integration with Google Maps Platform, competitive pricing in Southeast Asia, and comprehensive compliance certifications.

4.2 Cherry EV Telematics Integration

Deep integration with Cherry EV's vehicle telematics platform provides real-time vehicle data, diagnostics, and video monitoring.

Data Points Captured

  • Location: GPS coordinates at 3-second intervals
  • Speed: Current vehicle speed in km/h
  • Battery: State of charge (%), estimated range
  • Odometer: Total distance traveled
  • Diagnostics: DTC codes, health status
  • Video: Event-triggered clip URLs

Integration Protocol

// Cherry EV API Configuration

Endpoint: https://api.cheryev.com/v2/telematics
Auth: OAuth 2.0 Client Credentials
Protocol: REST + WebSocket (streaming)
Rate Limit: 1000 requests/minute

// Data Sync Strategy
- Batch polling: Every 30 sec for fleet status
- WebSocket: Real-time during active trips
- Webhooks: Event-driven alerts

Full compliance with Malaysia's Personal Data Protection Act 2010 (PDPA).

PDPA Principles Implemented:
  • General Principle: Data processed only with user consent
  • Notice & Choice: Clear privacy policy and consent mechanisms
  • Disclosure: No third-party sharing without explicit consent
  • Security: Technical and organizational measures to protect data
  • Retention: Data retained only as long as necessary
  • Access: Users can request and receive their personal data
